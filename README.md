# maisPraTiAtv07

# Projeto de Atividades de Codificação com React

## Descrição
Este repositório contém uma série de atividades de codificação com React, projetadas para praticar e aprimorar habilidades em desenvolvimento front-end. Cada atividade está encapsulada em um componente separado e pode ser acessada através de um menu principal.

## Estrutura do Projeto
A estrutura do projeto está organizada da seguinte forma:

```plaintext
src/
  components/
    Contador.jsx
    AlteracaoCorFundo.jsx
    ListaTarefas.jsx
    Temporizador.jsx
    FiltroLista.jsx
    FormularioRegistro.jsx
    RequisicaoDados.jsx
    GaleriaImagens.jsx
    TimerComAlerta.jsx
    TabsNavegaveis.jsx
  App.jsx
  main.jsx
  App.css

Propósito de Cada Atividade
1. Contador Simples
Descrição: Um componente de contador que permite incrementar e decrementar o valor.

Desafio: Impedir que o valor seja menor que zero.

Localização: src/components/Contador.jsx

2. Alteração de Cor de Fundo
Descrição: Um componente que muda a cor do fundo da página ao clicar em um botão.

Desafio: Gerar cores aleatórias a cada clique.

Localização: src/components/AlteracaoCorFundo.jsx

3. Lista de Tarefas (To-Do List)
Descrição: Um componente que permite adicionar, remover e marcar tarefas como concluídas.

Desafio: Filtrar para ver apenas as tarefas pendentes.

Localização: src/components/ListaTarefas.jsx

4. Temporizador com useEffect
Descrição: Um temporizador que conta o tempo em segundos.

Desafio: Adicionar botões para pausar e reiniciar o temporizador.

Localização: src/components/Temporizador.jsx

5. Filtro de Lista
Descrição: Um componente que renderiza uma lista de nomes com um campo de entrada para filtragem.

Desafio: Ignorar maiúsculas e minúsculas ao filtrar.

Localização: src/components/FiltroLista.jsx

6. Formulário de Registro Simples
Descrição: Um formulário com campos de nome, e-mail e senha.

Desafio: Validar que todos os campos estejam preenchidos antes de enviar.

Localização: src/components/FormularioRegistro.jsx

7. Aplicação de Requisição de Dados Simples
Descrição: Exibe uma lista de posts obtidos de uma API pública.

Desafio: Adicionar um botão para recarregar os dados e um indicador de carregamento.

Localização: src/components/RequisicaoDados.jsx

8. Galeria de Imagens com Visualização Detalhada
Descrição: Uma galeria de imagens com visualização ampliada em um modal.

Desafio: Adicionar um botão de “Fechar” no modal e navegação entre imagens.

Localização: src/components/GaleriaImagens.jsx

9. Timer com Intervalo e Alerta
Descrição: Um timer de contagem regressiva com alerta ao final.

Desafio: Adicionar botões para pausar e reiniciar o timer.

Localização: src/components/TimerComAlerta.jsx

10. Componentes com Tabs Navegáveis
Descrição: Uma interface de “tabs” onde cada aba exibe conteúdo diferente.

Desafio: Adicionar um efeito visual para destacar a aba ativa.

Localização: src/components/TabsNavegaveis.jsx

Menu Principal
O projeto possui um menu principal que permite a navegação entre as diferentes atividades. Cada atividade pode ser acessada separadamente clicando em seu respectivo link no menu.
